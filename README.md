DocEase is a simple full-stack web application where patients can book doctor appointments online.
Features :-

 User signup & login (basic authentication)

 Book appointments with available doctors

 Add and manage doctor details

 View appointment history

 Responsive UI for better user experience

 Stores data in MongoDB database

Tech Stack Used

Frontend:

HTML

CSS

JavaScript

React.js

Backend:

Node.js

Express.js

Database:

MongoDB

How to Run the Project
1️⃣ Clone the Repository
git clone https://github.com/your-username/DocEase.git
cd DocEase

2️⃣ Install Dependencies
For backend:
cd server
npm install

For frontend:
cd ../client
npm install

3️⃣ Start the Application
Start backend:
npm start

Start frontend:
npm run dev

Make sure MongoDB is running locally or update the connection string in the backend.
